3 ways to apply CSS

  1. Inline CSS
    • Using style=" " as attribute
    • used to apply unique style to Single Element
  2. Internal CSS
    • writing style tag under head section
    • Applies to single page
  3. External CSS
    • writing link under head section
    • we create seperate style sheetfile as a .css

Applying internal CSS

Selectors

  1. Name Selector == can apply by simply writing element name e.g.h3,p,a etc.
  2. # ID Selector == we need to pass that id to the element as id= " "
  3. . Class Selector== we need to write class in opening tag of that element as class=" "
  4. Group Selector== we can apply style to multiple elements e.g. table,tr,th,td {property: value}
  5. * Universal Selector== applies to all doccumnet. * { }

Color

  • Using name ==> Style="color:pink"
  • Using rgb ==> Style="color:rgb(205, 255, 208);"
  • Using #rrggbb Hex values ==> style="background-color: #5b5f5b;"
